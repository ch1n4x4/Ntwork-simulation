# NETWORK SIMULATION

## Objective

Scenario; I have recently been hired as a Network Security Engineer Intern at
CHAVEZ FINANCE.
The company has requested that I design and implement a secure small
enterprise network using Cisco Packet Tracer to simulate its branch office
infrastructure.

### Skills Learned

- Secure Network Infrastructure design & configuration
- Network Segmentation
- VLAN configuration
- Network Troubleshooting
- Application of The Principle of Least Privilege

### Tools Used

- Cisco Packet Tracer

### Process

- Network Design & VLAN Segmentation
- Inter-VLAN Routing
- ACL Configuration
- Troubleshooting and Testing
  
### Outcome 

Secure Network Design
